# Diwali Sales Analysis

## Project Overview
This project analyzes Diwali sales data to understand customer purchasing patterns and sales trends during the festive season.

## Objectives
- Analyze customer demographics
- Identify top selling product categories
- Understand purchasing behavior
- Generate business insights

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset contains customer details, product categories, gender, age group, and purchase amount.

## Key Insights
- Married women aged 26-35 contribute the highest purchases.
- Electronics and clothing are top-selling categories.
- Certain states show higher sales performance.
